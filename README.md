# IslandsOfAdventure
1、项目简述：本项目基于Unity 引擎 + C# 脚本开发的横版像素风格 2D 角色扮演（RPG）游戏《冒险岛》，完整实现了传统 RPG 游戏的核心游玩体系，包含多场景地图穿梭、角色动画战斗、数值成长、道具背包、技能养成、NPC 交互、任务悬赏等全套功能模块，拥有流畅的画面切换特效与完整的游戏逻辑闭环，全程使用 C# 编写业务脚本，面向对象设计玩家、敌人、道具、任务等实体类，代码模块化分层，便于后续新增怪物、技能、场景内容，是一套结构完整、可拓展性强的 2D RPG 成品


2、功能模块：
（1）.多场景地图与场景传送系统
（2）.完整动画骨骼系统
（3）.角色属性与等级成长体系
（4）.技能树养成系统
（5）.背包道具与商店系统
（6）.NPC对话交互系统
（7）.任务系统

目前部分UI设计还在进行；部分系统还在更加完善的进行中；麻雀虽小五脏俱全，已经玩命开发中，敬请期待🤯🤯🤯🤯🤯🤯.......
